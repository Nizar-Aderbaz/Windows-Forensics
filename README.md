# Windows Forensics Project

A simple, end-to-end Windows Forensics investigation covering memory forensics, disk imaging, registry analysis, and user activity reconstruction.

The project also includes analysis of key Windows artifacts such as MFT, USN Journal, AmCache, AppCompatCache, Prefetch, BAM, services, scheduled tasks, startup items, and event logs.

All collected evidence is combined into a Super Timeline to fully understand system and attacker activity.
